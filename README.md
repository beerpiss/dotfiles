# dotfiles

    beerpsi's dotfiles
    ------------------
    uses GNU Stow - https://www.gnu.org/software/stow/
    
    Usage 
    cd ~
    git clone https://github.com/beerpsi/dotfiles.git
    stow <config you want to use>
      ex: stow polybar, stow i3...
